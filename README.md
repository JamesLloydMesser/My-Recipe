	Run the following with Conda for either macOS/Linux or Windows:

	build.sh (for macOS/Linux):
		#!/bin/bash
		$PYTHON setup.py install

	bld.bat (for Windows):
		"%PYTHON%" setup.py install
		if errorlevel 1 exit 1
